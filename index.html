📋 What's New v1.4
February 2026 · Folder System & Companion Updates
✦ Main Feature
NEW Folder System — Organise snippets into named folders with a collapsible tree view. Click 📁 + in the Explorer header to create a folder. Use + inside a folder to add a snippet directly to it, or hover any snippet and click 📂 to move it via a context menu. Deleting a folder keeps all its snippets as loose files.
✦ New Features
NEW Undo / Redo — ↩ ↪ buttons in the toolbar. Ctrl+Z / Ctrl+Y / Ctrl+Shift+Z. Up to 200 snapshots per snippet, independent per file.
NEW Python Progress Bar — animated shimmer bar shows loading stages (Fetching runtime → Initialising → Loading stdlib → Complete) during Pyodide boot and execution.
NEW 🐢 Turtle Graphics — import turtle now works! Renders on an HTML5 Canvas inside the terminal panel. Turtle icon visible from the start, animates with heading changes. Supports 40+ turtle API functions.
NEW Smart Autocomplete — suggestions now include your own variables, parameters, function names, class names, loop vars, and import aliases — live-parsed as you type, shown with a cyan ✎ badge.
✦ Fixes & Changes
FIX Docs panel close button was hidden by flex overflow. Restructured header so ✕ is always visible. Escape key also closes the panel.
FIX Turtle canvas was too wide and overflowing the terminal panel. Now constrained with max-width/max-height and scales to fit.
REMOVED Trailing whitespace warnings removed from the Correction panel for Python, Lua, and JS — whitespace doesn't affect code correctness.
